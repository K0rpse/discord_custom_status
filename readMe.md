### Installation

_How to install it_

1. Clone the repo
   ```sh
   git clone https://github.com/K0rpse/discord_custom_status.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API in `config.js`
   ```js
   const API_KEY = "ENTER YOUR API";
   ```
